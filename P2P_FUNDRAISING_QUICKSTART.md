## ğŸ” Debugging Tips

### Page not loading?
1. Check console for errors: `F12` â†’ Console tab
2. Verify CSS file is included: `<link>`
3. Verify JS file is included: `<script>`
4. Check file paths are correct

### Styles not applying?
1. Hard refresh page: `Ctrl+Shift+R`
2. Check CSS file path: Should be `css/p2p-fundraising.css`
3. Check for browser cache
4. Verify no conflicting CSS

### JavaScript not working?
1. Check console for errors
2. Verify HTML elements exist with correct data attributes
3. Check JavaScript file order (CSS before JS)
4. Verify no JavaScript errors on page

### Data not showing?
1. Mock data is generated by default
2. Check browser console
3. Verify JavaScript class initialized
4. Look for `new ClassName()` in console

## âš¡ Performance Tips

### Tip 1: Lazy Load Images
```html
<img src="image.jpg" loading="lazy" alt="Description">
```

### Tip 2: Cache Data
```javascript
// Store in localStorage
localStorage.setItem('fundraisers', JSON.stringify(data));
const cached = localStorage.getItem('fundraisers');
```

### Tip 3: Minimize Reflows
```javascript
// Bad
for (let i = 0; i < 1000; i++) {
    el.style.width = i + 'px';  // Reflow each time
}

// Good
const style = el.style;
for (let i = 0; i < 1000; i++) {
    style.width = i + 'px';  // Single reference
}
```
