# Microbial Community Shifts in Polluted Habitats

## Issue #1594 Implementation

### Overview
This educational feature explores how pollution disrupts microbial communities at the foundation of aquatic and terrestrial ecosystems, affecting nutrient cycling, decomposition, and overall ecosystem health.

### Educational Objectives
- Understand the role of microbial communities in ecosystem functioning
- Learn how different pollutants affect microbial diversity and composition
- Explore the cascading effects on food webs and ecosystem services
- Examine remediation strategies for restoring microbial communities

### Technical Implementation

#### Files Created
- `MICROBIAL_SHIFTS_README.md` - This documentation file
- `frontend/pages/microbial-shifts.html` - Main HTML structure
- `frontend/css/pages/microbial-shifts.css` - Styling and animations
- `frontend/js/pages/microbial-shifts.js` - Interactive functionality

#### Interactive Features
1. **Pollution Impact Simulator**
   - Real-time modeling of microbial community responses to different pollutants
   - Visualization of diversity changes and functional shifts
   - Comparison of acute vs. chronic pollution effects

2. **Microbial Diversity Calculator**
   - Shannon diversity index calculations
   - Species richness and evenness analysis
   - Pollution tolerance indices

3. **Nutrient Cycling Simulator**
   - Nitrogen and phosphorus cycling disruptions
   - Decomposition rate changes
   - Carbon sequestration impacts

4. **Biomarker Analysis Tool**
   - Microbial stress indicators
   - Functional gene abundance changes
   - Recovery potential assessment

- **Microbial Ecology Basics**: Introduction to microbial roles in ecosystems

- **Detection Methods**: Molecular techniques, metagenomics, bioindicators
- **Restoration Strategies**: Bioremediation, bioaugmentation, natural recovery

### Scientific Foundation

#### Key Concepts
- **Microbial Diversity**: Alpha, beta, and gamma diversity in polluted environments
- **Functional Redundancy**: How microbial communities maintain functions despite diversity loss
- **Pollution-Induced Stress**: Cellular responses and community adaptations
- **Trophic Interactions**: Effects on higher organisms through microbial changes

#### Pollution Types Modeled
- **Heavy Metals**: Mercury, lead, cadmium toxicity and bioaccumulation
- **Organic Pollutants**: Pesticides, hydrocarbons, pharmaceuticals
- **Nutrient Pollution**: Eutrophication effects on microbial communities
- **Plastic Pollution**: Microplastics and associated chemical contaminants
- **Acid Rain**: pH changes and aluminum toxicity

#### Ecosystem Services Affected
- **Nutrient Cycling**: Nitrogen fixation, nitrification, denitrification
- **Decomposition**: Organic matter breakdown and carbon cycling
- **Primary Production**: Algal and plant-microbe interactions
- **Water Quality**: Pathogen control and toxin degradation
- **Soil Health**: Plant-microbe symbioses and nutrient availability

### Interactive Learning Tools

- **Input Parameters**:

- **Output Metrics**:
  - Diversity indices (Shannon, Simpson)

  - Recovery time estimates
